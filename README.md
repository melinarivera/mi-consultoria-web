# NovaFlow | Consultoría y Gestoría Digital

Proyecto web tipo startup desarrollado como parte de mi portafolio como **Fullstack Developer en formación**.

Este proyecto simula la web de una consultoría digital junto con un panel administrativo (dashboard), mostrando tanto la parte pública (landing) como una interfaz interna tipo producto SaaS.

## Objetivo del proyecto

Desarrollar una aplicación web estructurada que represente:

- Una **landing page profesional tipo startup**
- Un **dashboard administrativo estilo SaaS**
- Buenas prácticas de organización de código
- Base escalable para futuros proyectos reales

---

## Funcionalidades

### Landing Page
- Diseño moderno y responsive
- Secciones:
  - Hero
  - Servicios
  - Beneficios
  - Testimonios
  - CTA (Call to Action)
  - Formulario de contacto
- Animaciones al hacer scroll
- Navegación fluida

### Dashboard
- Sidebar de navegación
- Métricas (clientes, ingresos, crecimiento, etc.)
- Tabla de clientes
- Lista de actividades
- Sistema de tareas (checkbox)
- Simulación de actualización de datos
- Responsive (adaptado a móvil)

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
